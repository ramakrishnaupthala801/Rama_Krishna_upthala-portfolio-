Healthcare Data Lakehouse

Overview

This project builds a full healthcare data lakehouse using Azure ADF, Databricks, and Delta Lake. It processes patient, claims, and clinical datasets using a Bronze → Silver → Gold architecture.

Architecture

  • Azure Data Factory for ingestion pipelines
  • Databricks notebooks for ETL and transformations
  • Delta Lake for ACID storage and schema evolution
  • Azure Data Lake Storage for raw and curated zones
  • Power BI for analytics and reporting

Key Achievements

  • Improved data reliability with Delta Lake ACID transactions
  • Reduced ETL runtime by 40% using optimized Spark jobs
  • Delivered clean, governed datasets for ML and BI teams
