ML Feature Store for Churn Prediction

Overview

This project builds a scalable ML Feature Store using Spark and Python to support churn prediction models. It standardizes feature creation, improves data quality, and accelerates model training.

Architecture

  • Spark jobs for batch and streaming feature engineering
  • Delta Lake for storing historical and real-time features
  • Python scripts for feature transformations
  • Databricks for orchestration and versioning
  • ML models consuming curated features for churn prediction

Key Achievements

  • Improved feature quality by 35% using automated validation
  • Reduced model training time by 50% with pre-computed features
  • Enabled consistent features across training and inference
