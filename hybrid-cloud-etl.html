Hybrid Cloud ETL Platform

Overview

This project integrates AWS and Azure into a unified ETL platform using Airflow and Jenkins. It supports cross-cloud data movement, automated pipelines, and scalable transformations.

Architecture

  • Airflow for orchestration across AWS and Azure
  • Jenkins for CI/CD automation
  • S3 and ADLS for storage layers
  • Spark jobs for heavy transformations
  • Secure cross-cloud networking for data movement

Key Achievements

  • Reduced ETL failures by 60% with automated monitoring
  • Enabled seamless AWS ↔ Azure data exchange
  • Improved pipeline deployment speed using CI/CD
