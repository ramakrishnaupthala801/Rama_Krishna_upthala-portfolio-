Real-Time Retail Streaming Pipeline

Overview

This project delivers a real-time retail analytics pipeline using AWS Kinesis, Kafka, Glue, and Redshift. It processes streaming sales events and provides sub-second insights for dashboards and fraud detection.

Architecture

  • Kafka → Kinesis for ingestion
  • AWS Lambda for lightweight transformations
  • AWS Glue for ETL and schema evolution
  • Redshift for analytics and BI dashboards
  • S3 for raw and curated storage

Key Achievements

  • Reduced data latency from 15 minutes to under 10 seconds
  • Enabled real-time fraud detection alerts
  • Delivered analytics-ready datasets for BI and ML
